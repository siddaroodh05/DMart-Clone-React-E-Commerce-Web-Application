# DMart Clone – Complete React E‑Commerce Project

A fully functional DMart-style e‑commerce website built with React, featuring cart management, search, categories, product details, and responsive design.

## Features

- Full Cart System: add, remove, change quantity; localStorage persistence  
- Product Grid: responsive layout with hover effects  
- Category Navigation: dynamic sidebar categories from API  
- Search Functionality: real-time product filtering  
- Product Details: image gallery, pricing, add‑to‑cart  
- Mega Menu: full‑screen category browser with brands  
- Header Modals: cart preview, notifications, sign‑in  
- Responsive Design: mobile‑first, sidebar hidden on small screens  
- Loading States: skeleton loaders for better UX  
- Animations: smooth transitions, hover effects, carousel  

## Tech Stack

- React 18 + React Router DOM v6  
- CSS Grid / Flexbox layouts  
- DummyJSON API (200+ products)  
- React Hooks state management

## API Endpoints

- https://dummyjson.com/products  
- https://dummyjson.com/products/categories  
- https://dummyjson.com/products/search  

## Responsive Breakpoints

- Desktop: full sidebar + 4‑column grid (> 900px)  
- Mobile: no sidebar + single column (< 900px)
